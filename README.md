# 📖 Frequência EBD

Aplicativo PWA para controle de frequência da Escola Bíblica Dominical 2026.

## 🚀 Como instalar via GitHub Pages

### 1. Crie o repositório
1. Acesse [github.com](https://github.com) e faça login
2. Clique em **New repository**
3. Nome sugerido: `frequencia-ebd`
4. Marque como **Public**
5. Clique em **Create repository**

### 2. Faça upload dos arquivos
Envie todos estes arquivos para o repositório:
```
index.html
manifest.json
sw.js
icons/
  icon-192.png
  icon-512.png
README.md
```

### 3. Ative o GitHub Pages
1. No repositório, clique em **Settings**
2. No menu lateral, clique em **Pages**
3. Em **Source**, selecione `Deploy from a branch`
4. Em **Branch**, selecione `main` → `/ (root)`
5. Clique em **Save**

### 4. Acesse o app
Após ~1 minuto, o app estará disponível em:
```
https://SEU_USUARIO.github.io/frequencia-ebd/
```

### 5. Instale no celular
- **Android (Chrome):** Acesse o link → menu (⋮) → "Adicionar à tela inicial"
- **iPhone (Safari):** Acesse o link → compartilhar (□↑) → "Adicionar à Tela de Início"
